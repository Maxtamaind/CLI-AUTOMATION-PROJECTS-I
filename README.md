# CLI Automation Projects

This repository contains simple and practical CLI for automating every tasks using:

- **Bash** (Linux/maxOS)
- **PowerShell** (Windows)
- **Batch** (.bat for Windows)

  The goal is to build a growing collection of small automation tools while improving scripting skills.

  ---

  ## Repository Structure

  ## Completed Scripts

  ### **1. organize-downloads.sh**
  **Location:** `bash-scripts/`
  **Description:**
  Automatically sorts files in your Downloads folder into subfolders (Images, Documents, Archives).
